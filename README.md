## 1. Title
RoadSafe Analytics Road Accidents (India)

---

## 2. Problem Statement

Road accidents are a major public safety issue in India, causing loss of lives and property every year.

The objective of this project is to:
- Analyze road accident data across India
- Identify accident patterns and high-risk areas
- Understand key causes and contributing factors
- Provide data-driven insights to improve road safety.

This project aims to support better decision-making using data analysis.
