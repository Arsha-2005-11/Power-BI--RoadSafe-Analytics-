
### Data Analysis
Data Analysis means studying data to find useful information and support decision making.

### Data Visualization
Data Visualization means presenting data using charts, graphs, or dashboards so it is easy to understand.

### Data Transformation
Data Transformation means changing or cleaning data so it can be used properly for analysis.

### Introduction to Power BI
Power BI is a data analysis and visualization tool developed by Microsoft. It helps users analyze data and create reports and dashboards.

### Power Query
Power Query is a tool in Power BI used for importing, cleaning, and transforming data before analysis.
