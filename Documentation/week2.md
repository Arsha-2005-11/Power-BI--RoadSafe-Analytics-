Data Cleaning

1. Types of Data

Data can be classified into three main types:
 a) Numerical Data (Quantitative)
- Represents measurable quantities.
- Can be discrete (countable) or **continuous** (any value in a range).
- Examples:Age, Salary, Temperature, Number of cars sold

 b) Categorical Data (Qualitative)
- Represents groups or categories.
    Example: Gender (Male/Female)
 c) Time-Series Data
- Represents data points collected over time intervals.
- Used to analyze trends or patterns over time.
- Examples Daily stock prices, Monthly sales, Yearly rainfall

 2. Basics of Data Cleaning

Data cleaning is the process of **preparing raw data for analysis** by correcting or removing errors. Clean data ensures accurate results.

Common Steps in Data Cleaning

1. Handling Missing Data
   - Remove rows/columns with too many missing values.
   - Fill missing values with mean, median, mode, or placeholder.

2.Removing Duplicates
   - Identify and remove repeated records to avoid bias.

3.Correcting Errors
   - Fix typos, inconsistent formats, and outliers.
   - Example: "NYC" and "New York City" → standardize.

4. Standardizing Data
   - Ensure consistent units or formats.
   - Example: Dates (DD/MM/YYYY vs MM/DD/YYYY)

5. Handling Outliers
   - Outliers may skew analysis.
   - Decide whether to remove or adjust them based on context.

6. Converting Data Types
   - Ensure numbers, dates, and categories are in the correct type.

7. Validating Data
   - Check if values fall within expected ranges.
   - Example: Age should be > 0 and < 120
